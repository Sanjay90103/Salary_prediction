Key System Features

End-to-End ML Pipeline

Features numerical scaling and categorical encoding encapsulated in a unified scikit-learn Pipeline, preventing data leakage and simplifying prediction logic.

🛡️

Robust Validations

Includes dual-layer validations (client-side JS and backend Flask API) checking for invalid formats, boundary ranges, and logical constraints.

A modern user interface leveraging AJAX/Fetch API for seamless, page-reload-free interactions and responsive desktop/mobile layouts.
